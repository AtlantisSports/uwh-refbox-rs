2. Recent Enhancements

2.1 Dynamic Font Sizing System ✅ COMPLETE

Implementation: refbox/src/app/dynamic_font_sizing.rs

The dynamic font sizing system automatically adjusts font sizes to accommodate long text content while maintaining readability and layout consistency.

🎯 Key Features

  • Group-based Font Sizing: All referee information cells use consistent font sizes
  • Real-time Text Measurement: Uses Roboto-Medium.ttf font for accurate width calculations
  • Minimum Readability: Enforces minimum font size (12px) to maintain usability
  • State Management: Resets font sizes on game state changes

📐 Target Cells

  • Chief Ref: 176px available width
  • Timer: 176px available width
  • Water Ref 1-3: 176px available width each
  • Team Names (Last/Next Game): 196px available width

🔄 Font Sizing Algorithm

1. Measure text width at default size (18px)
   ↓
2. If text exceeds available width
   ↓
3. Use binary search to find optimal size
   ↓
4. Apply group-based sizing (all cells use smallest required size)
   ↓
5. Enforce minimum font size for readability (12px minimum)

9. Code Snippets

Font Sizing Test Example

#[test]
fn test_font_resizing_with_specification_test_data() {
    let test_cases = vec![
        ("Russell Owen Camilo La Torre", GameInfoCell::ChiefRef),
        ("Muhammad Danish Haikal Mohd Fadel", GameInfoCell::WaterRef2),
        ("A very long person name", GameInfoCell::WaterRef3),
    ];

    let mut dfs = DynamicFontSizing::new();
    for (text, cell) in test_cases {
        dfs.update_cell_font_size(cell, text);
        let font_size = dfs.get_font_size(cell);

        assert!(font_size >= MIN_FONT_SIZE);
        assert!(font_size <= SMALL_TEXT);

        let actual_width = measure_text_width(text, font_size);
        assert!(actual_width <= cell.available_width());
    }
}

Label Width Logic Test

#[test]
fn test_label_width_logic() {
    use iced::Length;
    let cases = vec![
        ("Last Game", Some(GAME_LABEL_WIDTH)),
        ("Next Game", Some(GAME_LABEL_WIDTH)),
        ("Chief Ref", Some(REF_LABEL_WIDTH)),
        ("Timer", Some(REF_LABEL_WIDTH)),
        ("Water Ref 1", Some(REF_LABEL_WIDTH)),
        ("Other", None),
    ];
    for (label, expected) in cases {
        let width = determine_label_width(label);
        match expected {
            Some(w) => assert!(matches!(width, Length::Fixed(x) if x == w)),
            None => assert!(matches!(width, Length::FillPortion(_))),
        }
    }
}
